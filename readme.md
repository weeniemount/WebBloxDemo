# Installation Guide
## This is for anyone who wants to run this offline, as of version 0.1 Demo (I literally js made up the version number)
## Step 1
Download the whole entire repo
## step 2
extract the zip file
## step 3
open up a terminal to the root of the project (where index.html is, not in the archive folder)
## step 4
install Python if not installed
## step 5
run the command:
if on windows: python -m http.server 8080
if on mac or linux: python3 -m http.server 8080
## step 6
open up any web browser and type this:
# http://localhost:8080
it should load WebBlox! Be sure to check back to this repository for updates and different instructions!
